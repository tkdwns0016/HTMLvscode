자바스크립트 이벤트

이벤트를 적용시키는 방법
  • 인라인 방식
    1. 태그 안쪽에 이벤트 속성을 이용하여 추가하는 방법
    2. 예) onclick="실행 할 함수", onfocus="실행 할 함수"
  • 객체의 이벤트 속성에 함수를 추가 하는 방법
    1. 요소.on이벤트명 = 실행 할 함수 정의
    2. 예) element.onclick=function(){alert(); }
  • addEvent함수를 이용하여 추가하는 방법
    1. 요소.addEventListener("이벤트명",함수정의)
    2. 예) element.addEventListener("click",function(){실행 할 문장; })
