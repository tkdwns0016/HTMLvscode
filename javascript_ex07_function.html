함수

함수를 선언하는 방법
  • function 함수명([param1[,param2[,...param]]]){내용} //대괄호는 형식을 표현하는 표현식에서는 생략가능을 의미한다.
    • 함수를 선언하면 함수가 호이스팅 됨
    • 어디서도 사용이 가능하다.
  • var x = function([param1[,param2[,...param]]]){내용}
    • 익명 함수
    • 변수 x가 생성된 시점 이후에 사용 가능하다.
    • 변수 x로 함수를 호출한다.
    • 예 : x(값1,값2,...)
  • (function([param1[,param2[,...param]]]){내용})(값1,값2)
    • 익명 함수, 즉시 실행 함수
