element 생성과 삭제

요소를 추가하는 방법
  1. innerHTML에 직접 문자열로 태그를 만들어 추가
    • element.innerHTML = "<li>추가하는 아이</li>"
  2. 요소 & 텍스트노드 생성 함수를 이용하는 방법
    • 요소 생성 : createElement("태그명")
    • 텍스트노드 생성 : creatTextNode("글자")
    • 노드 추가
      • appendChild(추가할 노드객체)
      • insertBefore(추가할 자식, 기존 자식)
노드 삭제 방법
  1. 태그로 직접 지움
    • innerHTML ="";
  2. 삭제 메서드 활용
    • 부모요소.removeChild(지울 자식객체)
    • 지우려는아이.remove()
  • 맛집!
