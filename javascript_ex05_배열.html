배열

배열을 생성
var myArray = [데이타1,데이타2,데이타3...];
var myArray = new Array(데이타1,데이타2,데이타3...);
var myArray = []; 빈배열 var myArray = new Array();
배열의 사용
배열변수명[index] = 넣을 값
myArray[0] = "값1";
myArray[1] = 100;
배열의 기본 속성
길이 : 변수명.length
배열의 특징
  • 데이타 타입이 서로 달라도 된다.
  • 배열의 길이가 가변이 된다.
  • 배열 자체에 기본 메서드도 있다.
