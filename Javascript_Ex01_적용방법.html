자바스크립트란?

자바스크립트
-스크립트언어 : 어플리케이션(다른 소프트웨어)을 제어하기 위한 언어.
-인터프리터 언어 : 컴파일을 하지 않고 한줄씩 해석해서 실행하는 언어.
-동적 타이핑(자료형) : 변수가 실행되면서 자료형이 정해짐
-함수형 프로그래밍언어 : 함수형태로 구현해서 사용하는 언어
적용방법
  1. 인라인방식
    • 태그 안쪽에 넣는 방법
    • 이벤트 속성과 같이 쓴다.
    • 클릭시 alert를 발생
  2. 내부스크립트 방식
    • 문서 안쪽에 script태그안에 넣어 쓰는 방식
    • 클릭시 alert를 발생
  3. 외부스크립트 방식
    • 파일을 외부에 만드는 방식
    • 확장자가 ,js인 파일로 따로 만들어 가져와 씀
    • 클릭시 alert를 발생
자바스크립트 문법
  • 한줄 단위로 문장이 끝난다.
  • 세미콜론(;)으로 끝낼 수 있다.
  • 블록 : 중괄호({)
  • 지역변수는 function단위로만 나뉜다.
  • 주석 : //내용 슬래시 두개, /* 내용*/
