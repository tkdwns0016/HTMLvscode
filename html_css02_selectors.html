CSS 선택자

css가 어떤 요소(element)를 선택할 것인가를 판단하기 위한 방식

태그로 선택하는 방식
형식 -> 태그명{css속성:값}

id로 선택하는 방법
형식 -> #id명{css속성 : 값}
class로 선택하는 방법
형식 -> .class명{css속성:값}
전체를 선택하는 방법
형식 ->*{css속성:값}
