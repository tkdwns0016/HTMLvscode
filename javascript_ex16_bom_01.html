BOM

window
  • 브라우저를 관리하는 객체
  • 생략이 가능하다.
  • 모든 함수, 객체, 변수는 윈도우 안에 들어감
  • 브라우저 화면에 관련된 속성과 기능이 있음

넓이와 높이 출력

screen
  • 스크린 화면관련 객체
  • 높이, 넓이, 사용가능한 높이와 넓이

넓이와 높이 출력

location
  • 위치 관련한 기능
  • 현재 페이지 url or path
  • 페이지 전환

주소 정보

history
  • 페이지의 방문기록
  • 뒤로가기, 앞으로가기
popup box
  • alert("메세지") : 경고창
  • confirm("메세지") : 확인 취소버튼이 있는 팝업창
  • prompt("메세지") : 입력할 수 있는 팝업창

timing
  • setTimeout(function, milliseconds) : 일정시간 뒤 동작
  • setInterval(function, milliseconds) : 일장시간 마다 동작
