자바스크립트 자료형

Number
수와 관련된 자료형
실수, 정수 모두 이 타입이다.
String
문자열 타입
자바의 문자열타입과 거의 같다고 보면 됨.
Boolean
참 거짓을 나타내는 자료형
//상수 const var4 = "이것은 상수"; var4 = "변경하려고 할때 변경할 수 없다."; console.log("var4",var4); 거짓 : false, 0, undefined, null
Array
배열 자료형
서로 다른 타입을 넣을 수 있다.
array 자체에 자바의 List와 비슷한 메서드가 있다
Null
참조값이 없을때
undefined
정의되지 않은 변수를 나타낸다.
object
객체 타입
